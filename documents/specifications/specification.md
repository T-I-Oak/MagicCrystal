# Magic Crystal システム仕様書

## 1. バージョン管理

### 1.1 バージョン表記
アプリのバージョンは、セマンティックバージョニング（Major.Minor.Patch）に準拠して管理します。
- 現在のバージョンは `Game.js` の `this.version` に定義されています。
- バージョン情報は、タイトル画面の左下にも表示されます。

### 1.2 バージョン更新ルール
機能追加、バグ修正、またはリソースの変更を行った際は、必ずバージョン番号を更新してください。
- 小規模な修正: パッチバージョンを上げる（例: 1.0.0 -> 1.0.1）
- 新機能の追加: マイナーバージョンを上げる（例: 1.0.1 -> 1.1.0）

## 2. キャッシュ無効化対応

ブラウザキャッシュによる古いスクリプトの実行を防ぐため、`index.html` での JS 読み込み時にクエリパラメータを付与しています。

### 2.1 実装方法
`index.html` 内の `<script>` タグの末尾に `?v=1.0.0` の形式でバージョンを指定しています。

```html
<script src="game/main.js?v=1.0.0"></script>
```

### 2.2 更新手順
バージョン（`Game.js` の `this.version`）を更新した際は、**必ず `index.html` 内の全てのスクリプトタグに付与されている `?v=X.X.X` の値も同じバージョンに書き換えてください。**

これにより、ブラウザは新しいファイルとして認識し、最新のスクリプトを確実に読み込むことができます。
