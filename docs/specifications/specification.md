# Magic Crystal 仕様書・ドキュメント一覧

Magic Crystal プロジェクトのドキュメントは、目的に応じて以下のファイルに分割されています。

## 1. 共通基盤
- **[ゲーム論理仕様書](./system_specification.md)**:
  実装に依存しない、ゲームのルール、物理、タイルの特性などの核心情報。
- **[技術実装仕様書](./technical_specification.md)**:
  現在の JavaScript 版における実装詳細、アセット構成、入力マッピングなど。

- **[入力システム仕様書](./input_specification.md)**:
  キーボード、タッチ、バッファリング、エッジ検出などの入力処理詳細。
- **[開発運用ルール](./development_rules.md)**:
  バージョン管理方法、キャッシュ対策、著作権表記規定などのエンジニア向けガイドライン。

## 3. ステージ設計
- **[ステージ設計資料 (level.md)](./level.md)**:
  各ステージのコンセプトや難易度の設計図。
